Action / effect library + findings · the value-side reference

The evaluator’s answer key: for every finding, what acting on it is worth and how sure we are. The agent never sees this. Each action passes the validity gate — it carries a real target outcome and a QALY-per-event conversion, never a surrogate. The two axes stay separate: evidence grade is a tag, never folded into the QALY magnitude. Illustrative scaffolding, sourced but not validated.

Pricing gaps & false-zero findings

A gap detector is a real test not yet in the pricing catalog (carried with a proposed code/price). A false-zero finding is one whose every detector is a gap — it earns nothing until a real test is added, so it’s flagged rather than silently scoring zero.
